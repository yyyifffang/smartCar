# 智能車車(SmartCar)
智能汽車是一種高度計算機化的汽車，具有計算泛在、直觀的人機交互和開放的應用平台。 在這裡，我們提出了一個先進的智能汽車演示平台，該平台具有透明擋風玻璃顯示屏和各種運動傳感器，駕駛員可以在增強現實中操縱各種適合汽車的應用程序。 與智能手機類似，司機可以根據自己的需求免費下載適合汽車的應用程序來定制他們的智能汽車。
